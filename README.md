# .unison

## Introduction

This is my personal Unison user directory for settings and profiles.

## Installation

To install, simply clone this repo to your `.unison` directory:

`git clone https://github.com/mkrupcale/.unison.git ~/.unison`.

## Usage

See the official [reference](https://www.cis.upenn.edu/~bcpierce/unison/download/releases/stable/unison-manual.html#reference) guide on how to set up Unison profiles and the various preferences available.
